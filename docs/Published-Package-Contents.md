`verify-release-package.ps1` requires the package to contain:

- App binaries (`app/LocalAIFactory.Web.dll`).
- ‚â• 4 knowledge packs (each with a `manifest.json`).
- ‚â• 5 database `.ps1` scripts.

---

## 4. What is EXCLUDED (and enforced)

The verifier **fails** if the package contains any of these:

- **Secrets / keys:** `secrets.json`, `*.pfx`, `*.pem`, `*.key` ‚Äî and the `keys/` Data Protection
  folder is never packaged.
- **Database files:** `*.mdf`, `*.ldf`.
- **Backups:** `*.bak`.
- **Model files:** `*.gguf`, `*.onnx`, `*.safetensors`.
- **Oversized doc/screenshot assets:** any file under `docs/` larger than **5 MB**.

Observed: `[ OK ] no secret/db/backup/model files in package` and `[ OK ] no oversized doc/screenshot
assets`. This mirrors the repo-hygiene guarantees of the security audit
(`docs/Final-Security-Audit-Report.md`).
