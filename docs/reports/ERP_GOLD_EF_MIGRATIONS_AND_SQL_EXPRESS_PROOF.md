## Honest limitations

- The migration was **generated and verified locally but NOT applied against a live SQL Express** in this environment — no SQL Express instance was present. `setup-sql-express.ps1` is written to handle that case with graceful failure and guidance.
- `InitialCreate` covers the **deterministic model only** (65 tables). The 5 local-LLM-proposed catalog modules are not part of this committed migration.
- No live migration-upgrade chain is exercised yet (single InitialCreate baseline).
